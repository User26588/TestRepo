# CheckParamTag

## EmptyParamTag

### Description

Empty tag 'Content\/Param' in Command '{commandId}'.

### Properties

| Name         | Value       |
| ------------ | ----------- |
| Category     | Command     |
| Full Id      | 10.3.2      |
| Severity     | Major       |
| Certainty    | Certain     |
| Source       | Validator   |
| Fix Impact   | NonBreaking |
| Has Code Fix | False       |

### Details

Command\/Content tag should contain a list of 'Param' tags. The 'Param' tags should refer to the id of an existing Param.
