# CheckPidAttribute

## NonExistingId

### Description

Attribute 'Response\/Headers\/Header@pid' references a non\-existing 'Param' with ID '{pid}'. HTTP Session ID '{sessionId}'. Connection ID '{connectionId}'.

### Properties

| Name         | Value       |
| ------------ | ----------- |
| Category     | HTTP        |
| Full Id      | 8.15.1      |
| Severity     | Major       |
| Certainty    | Certain     |
| Source       | Validator   |
| Fix Impact   | NonBreaking |
| Has Code Fix | False       |

### Details

Use this attribute to specify the id of an existing parameter where the response header value will be set.
