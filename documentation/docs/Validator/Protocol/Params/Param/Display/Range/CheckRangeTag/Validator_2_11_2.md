# CheckRangeTag

## UnsupportedTag

### Description

Unsupported 'Display\/Range' tag for '{paramDisplayType}' Param with ID '{paramPid}'.

| Name         | Value       |
| ------------ | ----------- |
| Category     | Param       |
| Full Id      | 2.11.2      |

| Fix Impact   | NonBreaking |

### Details
