# CheckParamTag

## EmptyParamTag

### Description

Empty tag 'Content\/Param' in Response '{responseId}'.

### Properties

| Name         | Value       |
| ------------ | ----------- |
| Category     | Response    |
| Full Id      | 11.3.2      |
| Severity     | Major       |
| Certainty    | Certain     |
| Source       | Validator   |
| Fix Impact   | NonBreaking |
| Has Code Fix | False       |

### Details

Response\/Content tag should contain a list of 'Param' tags. The 'Param' tags should refer to the id of an existing Param.
