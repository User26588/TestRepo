# CheckOptionsAttribute

## InvalidQActionAfterOption

### Description

Invalid value for 'qactionAfter' option. Expected format: 'qactionAfter:\<qactionId\>'. Current value '{optionValue}'.

### Properties

| Name         | Value       |
| ------------ | ----------- |
| Category     | Timer       |
| Full Id      | 7.3.15      |
| Severity     | Major       |
| Certainty    | Certain     |
| Source       | Validator   |
| Fix Impact   | NonBreaking |
| Has Code Fix | False       |

### Details

Expected format: 'qactionAfter:\<qactionId\>', where \<qactionId\> specifies the ID of the QAction.
